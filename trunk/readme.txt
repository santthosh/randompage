1. Introduction:

This plugin, add facilities to access an arbitrary page on the internet. Pages are absolutely random, no login is required. 
A random page can be accessed using the Accelerator + '/' (Example : Ctrl + / in a windows environment) or using the provided ToolBar icon. 

Note: You might have to customize your tool bar to add the icon (which shows an astreix).

2. Installation:

Use FireFox Add-ons manager to add the extension

3. Supported Versions:

Mozilla FireFox 3.0 and above

4. More information:

http://www.notionforge.com/RandomPage
